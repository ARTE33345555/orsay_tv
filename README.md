Orsay TV Core — custom firmware for old Samsung Orsay, LG NetCast, WebOS 1.0, and Tizen 1.0 TVs.
It transforms your old TV into a smart multimedia device with 2D/3D menus, games, casting, and an app store.

✨ Features

CastToScreen — cast your screen to other devices

HelloTV++ — enhanced interface: search, Home Assistant, OpenWeather, browser

OliStore — local app store for your TV

Game Center — play games with 2D/3D support, Magic Remote, and gamepads

TV2D / TV3D / TV_3D.Stereo — switch between 2D and 3D interfaces

TV_UI / TV_Remote / TV_Camera — control the UI, remote, and camera

PyChromium Plugin — ad blocking, Flash support, and browser casting

All features work out of the box, directly on your TV! 😎

💻 Supported Devices

Samsung Smart TV (2010–2014) — Orsay / Tizen 1.0

LG NetCast / WebOS 1.0

Magic Remote and compatible remotes

Gamepads for gaming

⚡ Installation

Turn on your TV and open Smart Hub

Enable Developer Mode

Install the firmware:

Via USB

Via SSH / RC Upload (Visual Studio Code + RC Upload plugin)

For browser features, install PyChromium Plugin

🎮 Usage

Game Center: open with Magic Remote → select game → OK

Achievements: view by swiping left/right

2D / 3D menus: switch in TV settings

CastToScreen: start casting to any compatible device
